The dataset contains the following three directories:
	1. Train_docs
	2. Train_catches
	3. Test_docs

The first folder ("Train_docs") contains 100 case statements.

The second folder ("Train_catches")  contains the gold standard catchwords for each of these 100 case statements.

The third folder ("Test_docs") contains 300 test case statements. For each of these 300 statements, one should produce a set of catchphrases.
